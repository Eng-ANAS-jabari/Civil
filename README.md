# Conference Management App

تطبيق إدارة المؤتمرات مع Firebase

## المميزات
- عداد تنازلي للمؤتمر
- جدول زمني للجلسات
- معرض صور
- لوحة تحكم للمدير

## التقنيات المستخدمة
- React.js
- Firebase (Auth, Firestore)
- Tailwind CSS
- Lucide Icons

## التثبيت
1. `npm install`
2. أنشئ ملف `.env` وأضف إعدادات Firebase
3. `npm start`

## النشر
`npm run build`
